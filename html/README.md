# 🌊 Harbour & Hills Design System (App)

This is the Vite + React workspace for the **Harbour & Hills Design System**. 

Refer to the main [README.md](../README.md) in the root directory for full architectural documentation, design tokens, and components index.

## Quick Dev Commands

Install dependencies:
```bash
npm install
```

Start Vite dev server:
```bash
npm run dev
```

Build client for production:
```bash
npm run build
```
